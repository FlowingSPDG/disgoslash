## Getting Started

### Prerequisites
* [Golang](https://golang.org/dl/)
* [Discord](https://discord.com/)
    * Server `Guild ID`
* [Discord Application w/ Bot](https://discord.com/developers/applications)
    * Application `Public Key`
    * Application `Client ID`
    * Bot `Token`
